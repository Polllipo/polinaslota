# 🎨 Поліна Сльота — галерея дитячого мистецтва

Персональний сайт-галерея маленької художниці **Поліни Сльоти** (8 років, Горішні Плавні, Україна).
Тут зібрані її малюнки: котики, дракони, замки та цілі казкові світи, намальовані фарбами й олівцями.

🌐 **Сайт онлайн:** https://polllipo.github.io/polinaslota/

## Про проєкт

Це односторінковий сайт, який складається з **одного файлу** — `index.html`.
Усі зображення вбудовані прямо в код, тож жодних додаткових тек чи файлів не потрібно.

Розділи сайту:
- **Галерея** — 16 робіт у сітці по 3 в ряд, з переглядом кожного малюнка у великому вікні.
- **Про мене** — коротка розповідь про художницю та її фото.
- **Контакти** — зв'язок через маму, Telegram і телефон.
